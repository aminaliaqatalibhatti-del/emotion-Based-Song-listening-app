# 🎵 Emotion-Based Song Listener

An AI-powered music application that recommends songs based on the user's emotions. Instead of manually searching for music, users simply select or detect their current mood, and the application generates a personalized playlist that matches how they feel.

---

# Overview

Emotion-Based Song Listener is designed to make music discovery more personal and engaging. By analyzing the user's selected emotion, the app intelligently recommends songs that complement or improve their mood, creating a seamless and enjoyable listening experience.

The application features a clean, responsive interface and provides fast, personalized recommendations while delivering a smooth user experience.

# Installation

Clone the repository:

```bash
git clone <repository-url>
```

Navigate to the project:

```bash
cd emotion-based-song-listener
```

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

For the backend:

```bash
cd server
npm install
npm start
```

---

# Environment Variables

Create a `.env` file and configure the following variables:

```env
PORT=
MONGODB_URI=
```

# How It Works

1. Open the application.
2. Select or detect your current emotion.
3. The recommendation engine analyzes your mood.
4. Matching songs are retrieved.
5. Listen, explore, and save your favorites.
